Український веб-довідник
Веб-довідник створено, щоб допомогти Вам здобути нові знання, а також ефективніше використовувати вже.
Його мета - щоб будь-хто з легкістю зміг увійти в світ веб-розробки та без проблем створив свій перший сайт!
Ми та наші друзі - пул українського співтовариства, котрий буде постійно працювати, задля покращення українського сегменту інтернету та виведення його з теперішнього, незадовільного, стану. Тому, всі ми відкриті для нових ідей і готові до глобальних змін!

CSS

HTML
Мова розмітки гіпертексту (англ. HyperText Markup Language - HTML).
Основна мова для опису змісту веб-сторінок.

JS приклад:

JS
JavaScript (скорочено - JS) - це високорівнева мова програмування, яка використовується для надання динамічної поведінки веб-сторінкам. Дозволяє створювати інтерактивні елементи, анімації, змінювати вміст сторінок без їх перезавантаження, може маніпулювати об'єктами та подіями на веб-сторінці, змінюючи їх властивості.
Javascript - клієнтська мова програмування й виконується в браузері користувача, але в останні роки JavaScript також отримав можливість виконуватися на серверній стороні за допомогою середовищ, таких як Node.js чи подібних.
